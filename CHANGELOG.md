## [1.0.0] - 2026-02-24

### Added
- Initial public release of ContentFlow CMS monorepo (Next.js 16 + React 19 frontend, Express + MongoDB backend).
- Multi-tenant CMS admin and tenant site rendering with App Router layout.
- Core content modules and APIs for pages, blog, menu, footer, forms, media, SEO, themes, and tenant management.
- Authentication and OAuth endpoints, plus session/JWT-based server auth.
- External content API with rate limiting and domain extraction middleware.
- Activity logging, notifications, integrations, and API key endpoints.
- Deployment pipeline with CI checks, artifact packaging, PM2 release directories, health checks, and rollback.
